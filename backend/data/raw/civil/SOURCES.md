# Civil domain — source provenance

Raw bare-act text curated for TASKS.md T06. Each file was exported from the
official India Code portal PDF, converted to UTF-8 plain text, and normalized
to `Section N.` headers for the section-aware parser (T11).

Note: CPC *Orders* (Order I–LI) are excluded from this pass so rule numbers inside
each Order are not mislabeled as duplicate `Section 1.` headers; only operative
Sections 1–158 are retained.
